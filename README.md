
# final project ADA

## Project Description
This repository includes the BRFSS 2021 dataset, binary logistic regression model, and R markdown code used to analyze it. The project is part 
of an analysis of weight bias as it is associated with having a ehalthcare provider.
## Files Included
- `BRFSS 2021`: BRFSS 2021 dataset
- `ADA_FinalProject_code.rmd`: R script used to summarize and visualize the data
- `README.md`: This file

## What the Code Does
-loads needed packages
- Reads in the survey dataset
- Performs basic data cleaning and filtering
- Creates descriptive statistics table
- runs logistic regression model
- checks assumptions for logistic regression

## How to Run the Code
1. Download or clone this repository to your computer
2. Open `ADA_FinalProject_code.rmd` in RStudio
3. Make sure your working directory is set to the folder where the files are saved
4. Run the script
